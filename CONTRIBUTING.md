# Contributing to Gemini Care Assistant

Thank you for your interest in contributing to this project!

This repository contains a prototype for an AI-powered IoT activity monitoring system.

## How to Contribute

1. Fork the repository.
2. Create a new branch for your feature or fix.
3. Make your changes.
4. Commit your changes with a clear message.
5. Submit a Pull Request.

## Development Setup

1. Clone the repository:

git clone https://github.com/prachi01645/gemini-care-assistant.git

2. Install dependencies:

pip install -r requirements.txt

3. Run the project:

python src/main.py

## Guidelines

* Write clear and readable code
* Add comments where necessary
* Keep commits small and descriptive
* Test your code before submitting

## Reporting Issues

If you find a bug or want to suggest a feature, please open an issue in the repository.
